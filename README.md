# STEAM development kit hardware repository
This is the hardware design files of the STEAM development kit. This project was design in Altium Designer.
There is 2 folder:
- ESP32S3 Controller: the MCU board of the development kit, a seperate board that can operate independently
- Steam Dev kit: The main development kit with all of the peripherals, sensor, LEDs
